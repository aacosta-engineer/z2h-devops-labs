## 📌 About This Repository

This repository documents my hands-on learning journey through the **Z2H Academy DevOps Roadmap**.

The goal is to apply DevOps concepts through practical labs, exercises and projects while building experience with modern tools and cloud-native technologies.

Each section will contain practical implementations, configuration files, scripts, infrastructure code and documentation created throughout the learning process.

## 🗺️ Learning Roadmap

The repository will progressively cover the following areas:

- 🐧 Linux, Git & Bash
- 🐳 Docker & Docker Compose
- ☁️ AWS Cloud Infrastructure
- 🏗️ Infrastructure as Code with Terraform
- ☸️ Kubernetes & Helm
- 🔄 CI/CD & GitHub Actions
